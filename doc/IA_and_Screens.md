# 정보구조와 화면 목록 v1

이 문서는 화면 구조와 IA를 설명하는 요약 문서다. 로딩, 빈 상태, 오류 상태, 모바일 우선 규칙은 `UI_State_Spec.md`를 기준으로 한다.

## 1. 사이트맵

```text
홈 (/)
├── 번호 생성기 (/generate)
├── 회차 조회 (/draws)
└── 기본 통계 (/stats)
```

## 2. 네비게이션

* 헤더: 홈, 생성기, 회차 조회, 통계
* 푸터: 서비스 설명, 데이터 출처 안내

## 3. 화면 정의

### 홈 `/`

* 서비스 소개
* 주요 전략 요약
* 최신 회차 요약
* 생성기 이동 CTA

### 번호 생성기 `/generate`

* 전략 선택
* 생성 세트 수 선택
* 보너스 포함 여부 선택
* 생성 결과와 근거 표시

### 회차 조회 `/draws`

* 최근 회차 카드 목록
* 각 회차의 번호, 보너스, 추첨일 표시

### 기본 통계 `/stats`

* 번호별 출현 빈도 목록
* 전체 기준과 최근 10회 기준 전환

## 4. 공통 UI

* 상단 네비게이션
* 번호 볼 컴포넌트
* 카드 레이아웃
* 로딩 및 에러 상태
