# Product Spec v1.3

## 1. 제품 정의

Lotto Maker Lab은 과거 당첨 데이터 기반 추천 번호, 회차 조회, 통계, 회차 분석을 제공하는 무료 로또 웹 서비스다.  
핵심은 단순한 랜덤 생성기가 아니라, 사용자가 번호를 생성하고 결과를 해석하고 다시 방문하게 만드는 데이터 중심 경험이다.

현재 제품 목표는 다음 두 축으로 구성된다.

- 추천과 분석: 번호 생성, 통계, 회차 분석
- 생성 통계: 이 사이트에서 만든 번호들이 이번 주 당첨번호와 얼마나 맞았는지 공개 집계

생성 통계의 원본 저장소는 Firebase Firestore를 사용한다.

## 2. 핵심 사용자

### 2-1. 일반 사용자

- 빠르게 번호를 추천받고 싶은 사용자
- 최신 당첨번호와 최근 흐름을 보고 싶은 사용자
- 추천 결과를 참고용으로 가볍게 즐기는 사용자

### 2-2. 데이터 탐색형 사용자

- 자주 나온 번호, 덜 나온 번호, 최근 흐름을 보는 사용자
- 특정 번호나 패턴을 비교하고 싶은 사용자
- 생성 방식별 결과 차이를 보고 싶은 사용자

### 2-3. 재방문 사용자

- 매주 추천 결과를 다시 확인하는 사용자
- 이번 주 생성 통계와 전략별 성과를 보고 싶은 사용자
- 회차 분석과 최신 결과를 반복적으로 확인하는 사용자

## 3. 핵심 가치

1. 데이터 기반 추천을 빠르게 제공한다.
2. 최신 회차와 통계를 함께 보여준다.
3. 회차 분석과 생성 통계를 통해 재방문 이유를 만든다.
4. 복잡한 회원 커뮤니티가 아니라, 결과 기반 공개 집계를 제공한다.

## 4. 제품 구조

### 4-1. 홈

- 최신 당첨번호
- 빠른 추천 CTA
- 추천/통계/분석/생성 통계로 이동하는 허브
- 서비스 소개, 이용 방법, FAQ 요약, 확률 안내

### 4-2. 추천기

- `mixed`, `frequency`, `random`, `filter` 전략 제공
- 추천 결과와 근거 표시
- 번호 복사와 상세 통계 이동 지원
- 퍼블리시 후 추천 결과를 Firestore에 저장

### 4-3. 회차 조회

- 전체 회차 탐색
- 회차 상세 진입
- 번호 포함 회차 탐색

### 4-4. 통계

- 전체/최근 10회 비교
- hot/cold 요약
- 홀짝, 합계, 연속번호 요약
- 번호 상세 통계 이동

### 4-5. 회차 분석

- 최신 회차 분석 허브
- 회차별 기사형 분석
- 최신 결과, 회차 상세, 통계와 연결

### 4-6. 생성 통계

`생성 통계`는 기존 커뮤니티 개념을 대체한다.  
사람들이 글을 쓰고 댓글을 다는 공간이 아니라, 이 웹사이트에서 생성된 번호의 공개 집계 결과를 보여주는 기능이다.

핵심 구성:

- 이번 주 총 생성 수
- 최근 생성 조합 일부
- 전략별 생성 수
- 전략별 최고 적중 개수
- 이번 주 당첨번호와의 적중 분포

## 5. 사용자 플로우

### 5-1. 추천 중심 플로우

홈 → 추천기 → 결과 확인 → 번호 상세 통계 → 최신 결과/회차 분석

### 5-2. 통계 중심 플로우

홈 → 통계 → 번호 상세 통계 → 회차 상세 → 회차 분석

### 5-3. 재방문 플로우

홈 → 생성 통계 → 이번 주 전략 성과 확인 → 다시 추천기 사용

## 6. 상세 기능 요구사항

### 6-1. 추천기

- 추천 전략 선택 가능
- 기본 추천은 `mixed`
- 필터 추첨기는 다음 입력을 지원한다.
  - 고정수
  - 제외수
  - 홀짝 조건
  - 합계 범위
  - 연속번호 허용 여부
- 결과에는 번호, 보너스 여부, 전략, 생성 근거가 포함된다.

### 6-2. 생성 통계

- 생성된 번호는 익명 기준으로 저장한다.
- 저장 항목:
  - 번호 6개
  - 전략
  - 생성 시각
  - 필터 옵션
  - 익명 식별자
- 저장소:
  - Firebase Firestore
- 매주 최신 당첨번호가 반영되면 이번 주 생성 결과와 자동 매칭한다.
- 화면에서는 다음을 보여준다.
  - 총 생성 수
  - 전략별 생성 수
  - 전략별 적중 분포
  - 최고 적중 기록
  - 최근 생성 결과 일부

### 6-3. 회차 분석

- 최신 회차를 중심으로 분석 허브를 제공한다.
- 회차 상세와 최신 결과 페이지에서 분석으로 진입 가능해야 한다.
- 분석 페이지는 검색 유입용 랜딩 페이지 역할도 한다.

### 6-4. 정책/신뢰

- 개인정보처리방침
- 이용약관
- FAQ
- 문의/운영 안내
- 광고 정책 안내

## 7. SEO 및 유입 요구사항

### 7-1. 랜딩 페이지

- `/latest-lotto-results`
- `/lotto-number-generator`
- `/lotto-statistics`
- `/hot-numbers`
- `/cold-numbers`
- `/odd-even-pattern`
- `/sum-pattern`
- `/recent-10-draw-analysis`
- `/draw-analysis/[round]`

### 7-2. 검색용 설명 콘텐츠

- 생성기와 랜덤 추천의 차이
- 최근 20회 자주 나온 번호
- 홀짝/합계 해석
- 회차별 분석

## 8. 퍼블리시 전 완료 기준

- 추천기, 회차 조회, 통계, 회차 분석이 정상 동작
- 생성 통계 화면의 기본 구조와 공개 집계가 동작
- Firebase Firestore 저장 흐름이 동작
- 정책/신뢰 페이지가 모두 연결
- 모바일과 데스크톱에서 핵심 동선이 깨지지 않음
- Search Console 제출 준비 완료

## 9. 후속 확장

- 번호 궁합 통계
- 고급 패턴 통계
- 오늘의 운세 기반 추천 번호
- 그룹/반자동 조합 추천
