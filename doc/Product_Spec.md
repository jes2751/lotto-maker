# Product Spec v1.4

## 1. 제품 정의

Lotto Maker Lab은 과거 당첨 데이터 기반 추천, 회차 조회, 통계, 생성 통계를 제공하는 무료 로또 웹 서비스다.

핵심 포지션:
- 최신 결과를 빠르게 확인할 수 있어야 한다.
- 번호를 바로 생성할 수 있어야 한다.
- 숫자 흐름과 생성 결과를 참고용으로 볼 수 있어야 한다.

## 2. 핵심 사용자 질문

사용자가 가장 먼저 궁금해하는 질문은 아래 순서로 본다.

1. 이번 회차 당첨번호가 뭐지?
2. 지금 번호 하나 뽑아볼까?
3. 어떤 번호가 자주 나왔지?
4. 이번 주 사람들은 어떤 방식으로 번호를 만들었지?

제품 구조와 홈 구성은 이 질문 순서를 따라야 한다.

## 3. 핵심 페이지

### 3-1. 홈
목표:
- 최신 결과 확인
- 번호 생성 시작
- 핵심 통계 요약 확인

구성:
1. 최신 당첨번호 카드
2. 번호 생성 CTA
3. 핵심 통계 요약
4. 생성 통계 / 회차 분석 / FAQ 링크

홈에서는 설명형 콘텐츠와 정책 링크를 최소화한다.

### 3-2. 번호 생성기
목표:
- 사용자가 가장 빠르게 번호를 생성할 수 있어야 한다.

필수 요소:
- 전략 선택
- 생성 수 선택
- 필터 추천 옵션
- 생성 결과
- 생성 통계 반영 상태

### 3-3. 회차 조회
목표:
- 전체 회차 탐색
- 특정 회차 검색
- 회차 상세 이동

### 3-4. 통계 대시보드
목표:
- 전체 회차와 최근 10회 기준 핵심 흐름 비교
- 많이 나온 번호, 적게 나온 번호, 홀짝, 합계, 연속번호 요약 제공

### 3-5. 생성 통계 허브
목표:
- 게시판이 아니라 공개 생성 현황을 보여준다.

필수 요소:
- 이번 회차 생성 수
- 전략 점유율
- 전략 성과 보드
- 최근 생성 번호
- 적중 분포

## 4. 추천 전략

- `mixed`
- `frequency`
- `random`
- `filter`

기본 노출 원칙:
- 혼합 추천과 필터 추천을 중심으로 안내한다.
- 랜덤 추천은 비교용 보조 전략으로 유지한다.

## 5. 생성 통계 정의

생성 통계는 사람들이 소통하는 커뮤니티가 아니다.  
이 사이트에서 생성된 번호를 익명으로 저장하고, 회차 결과와 자동 매칭해 전략별 성과를 보여주는 공개 데이터 기능이다.

보여줘야 하는 값:
- 이번 회차 총 생성 수
- 전략별 생성 수
- 전략별 최고 적중 수
- 3개 이상 / 4개 이상 일치 수
- 많이 생성된 번호

## 6. 퍼블리시 전 필수 범위

- 홈 정보 밀도 축소
- 번호 생성기 완성
- 통계 대시보드 정리
- 생성 통계 허브 완성
- 정책 / 신뢰 페이지 정리
- Search Console 제출 준비
- 모바일 QA

## 7. 제외 범위

현재 v1에서는 아래를 제외한다.
- 자유게시판
- 댓글
- 좋아요
- 로그인 기반 프로필
- 회원 전용 기능

## 8. 후속 확장

- 번호 궁합 통계
- 고급 패턴 통계
- 오늘의 운세 기반 추천 번호
- 그룹 추첨기
